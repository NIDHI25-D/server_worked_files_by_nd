Setu Send Receipt By Email
-----------------------------

### USAGE

* Send Receipt From Journal Entry.
* Send Vendor Receipt By Email.

### Vendor Payment

* Send Vendor Receipt By Email.
* There is one drop down that if the payment is paid state and then, if you attach a file, it will send the mail to the
  vendor automatically.

### Journal Entry

* Send Receipt From Journal Entry.
* Into Journal Entry, you can find one action that sending the email with the cfdi document and receipt.
