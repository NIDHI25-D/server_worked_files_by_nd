Warehouse Efficiency Report
-------------------------

##### USAGE
* This module are creates a Efficiency Report for Warehouse and it's Picking.
* The main use of the report is it gives a time difference between Effective Date and Scheduled Date.
* Other values for the pickings mentioned in this report is **Qty Ordered Total**, **Qty Delivered Total**, **Assortment Start Date**, **Picking End Date**, takes a time between both of them as a **Total Time**, **Owner**, Picking name as a **Picking reference**, it's **Assortment Warehouse** etc.
* There are a path to locate the report is -
*       Inventory -> Reporting -> Picking Efficiency Report
*       Inventory -> Reporting -> Warehouse Efficiency Report
