Update Cost - Sale order line
-----------------------------

### USAGE

* This module is used to update the price of the products.
* If the price of product is changed, then in the mentioned Sale Order which includes respected Products, their cost price will be updated.
  

### CONFIGURATION
* To access this feature :
#####           Sale > Orders > Recalculate Sale Order Cost
* Kindly insert the From date and To date manually.
* Sale orders, appear between these dates will be seen in the Sale Orders field.
* To update the cost price in sale order lines of mentioned sale order, kindly continue the procedure with **Change** button
