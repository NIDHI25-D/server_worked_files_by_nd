Setu Auto Fillup Detailed Operation

USAGE
  • This module is used to create records of the products in Transfer, which are mentioned in the csv file.
  • This module only accepts file with format : ".csv"
  • This module is mainly used when user wants to add multiple products in a Picking.
Configuration
  • here are necessary columns are in CSV file to creates records are as below :
  1. reference_name
  2. product_sku
  3. destination_location
  4. quantity
  5. lot_serial_number

  • here, upload the file in Auto Fillup Detailed Operation wizard which is located at:

  • Inventory --> Configuration --> Auto Fillup Detailed Operation

  • To import the file, kindly click on Import button. This will import the file in respected Transfer.

  • Products mentioned in this file are reflected in respected Transfers.
