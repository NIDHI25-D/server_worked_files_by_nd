Estafeta Odoo Integration
-----------------------------

### USAGE

* This module is used to Establish the Estafeta to odoo integration.
* If it uses the estafeta, then fetch the charges from the estafeta for the delivery.

### Company

* Establish the integration of estafeta for company.
* Configure the api,api key, url, and all other configurations need to integrate estafeta.

### Delivery Methods

* Configure to start with estafeta service.
* Which delivery type is estafeta that only can access the estafeta configurations.

### Shipping Carriers

* To start with estafeta in Shipping Carriers, then tick the Use as estafeta rate.
* Configure for the Package and Pallet Information.
