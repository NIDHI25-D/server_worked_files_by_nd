Sales Time Dashboard
-----------------------------

### USAGE

* This module is used to create data of sales i.e - time difference between create date - confirmation date and other difference of sale flow till invoice validation.
  

### CONFIGURATION
* To access the sales time dashboard make sure user has group - Sales Time Report:
* also Sales Time Report group user are enabeling Re-calculate Sales Time Server action in sale order
* there are the group Customer Confirmation if it is active only then Customer Confirmation button are visible and use in sale order.
* here time calculation are calculate from resource calender where Default work shift ? are checked.
