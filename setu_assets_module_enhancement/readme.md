Setu Assets Module Enhancement
------------------------------

##### USAGE
* Used for show the **Other Info** tab in Accounting Assets.
* Added a field in asset models name **Is it computer equipment**.
* Creates main menu name **Asset Characteristics** in accounting module.
* From this menu **Asset Characteristics** there are various sub menus are available and it's relation are given to the assets under the tab **Other Info**.
* If Uninstall this module it will deletes all the records of other info page values & Also removes the value from the all menu under the main menu **Asset Characteristics**.
    
##### Configuration
* Here are the group name **Create Asset Equipment** if this group are given to the respective user only those types of users are enable to see, creates and edit in computer equipment.
* If Is it computer equipment field is set as True then only displays other info tab in assets while selecting the asset models.
