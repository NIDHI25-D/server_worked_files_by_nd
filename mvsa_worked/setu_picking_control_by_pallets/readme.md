Setu Picking Control By Pallets
-------------------------------

### USAGE
=========
* To create Massive Packages. 
* To Print report of Parcel Tracking Label.
* Create Massive package from barcode delivery.

Configuration
=============
* Set User Configuration:
### Users > manage packages(Boolean To show "put in pack" button in Transfer).

Needed Things
=============
* Required carrier id: 
### to set package type in new created packages, first set field carrier id in picking.

* Required to Create Massive Packages:
### To create massive packages, In product "package type" Boolean enabled is needed at product level.
### package type field is in purchase page so if purchase module is installed then after package type field is visible.
