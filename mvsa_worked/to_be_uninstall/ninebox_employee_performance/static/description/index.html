Performance (NineBox Employee Performance)

- This module helps you to Show NineBox report on Employees based on their Potential and Performacne Ratio - Review Employee Performacne using Employee and Manager Assessment Questions

Installation

- Copy ninebox_employee_performance module to addons folder

- Install the module normally like other modules

User Access

Settings --> Users&Companies --> Users --> Other --> Ninebox Employee Performance



Configuration

1. Create Review Schedule

Performance --> Performance Settings --> Review Schedule --> Create



Select Year and Review Period Frequency



Create Assesment Period



2. Create Employee Assessment Questions for Created Period

Performance --> Performance Settings --> Employee Assessment Questions --> Create



Create Employee Assessment Questions



3. Create Manager Assessment Questions for Created Period

Performance --> Performance Settings --> Manager Assessment Questions --> Create



Create Employee Assessment Questions



4. Initiate Employee Performance

Performance --> Performance Settings --> Review Schedule --> Assesment Period



Open Assesment Period --> Initiate Employee Performance



Initiated Employee Performance and State Changed



5. Employee Review

Performance --> Performance --> Employee Performance --> Select a Performacne Review record



Update your Assesment Rating and Comment for each Lines



Submit your Review to Manager



5. Manager Review

Performance --> Performance --> Employee Performance --> Select a Employee Review record for Update Manager Comments



Update Manager Assesment Rating and Comment for each Lines



Complete the Review



6. Ninebox Report

Performance --> Ninebox Report --> Ninebox Report --> Select a Period



Performance --> Ninebox Report --> Ninebox Report --> All Emplolyees List



Performance --> Ninebox Report --> Ninebox Report --> Profile



7. Print Ninebox Report

Performance --> Ninebox Report --> Ninebox Report --> Print Ninebox Report
