Setu Product Price Update
-------------------------

##### USAGE
* This module is used to updating the prices of the products according to
the discount levels that are defined in the product catalog.
* This module, allows updating the prices of the products based on the formula.
* This module even includes the feature of changing the price of product to previous price after updating a new price.
* Overall, this module is used to update the price of the products mentioned in the Price Update with applied discounts, import factor as well as exchange rate.
* It will raise respected warning if update price is less than current price.

##### CONFIGURATION
* There are the various fields at configuration level i.e - Level, Range1, Range2, Import Factor, Responsible, Exchange rate, Discounts
* This fields are use to set default values to the Product Price Update.
