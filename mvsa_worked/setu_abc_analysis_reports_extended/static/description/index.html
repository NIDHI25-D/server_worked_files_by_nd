ABC Sales Analysis Reports Extended


USAGE :
  • This module is extended version of ABC Sales Analysis Reports.
  • This module, includes a cron(scheduled action) named : Product Classification.
  • This module is used to upgrade the classification per Products in Product model as per the sale of the respected products.
  • Classification of the Product is depended on duration of the time mentioned in the Settings.

CONFIGURATION : 
  •  Settings -> Quotations & Orders -> Product Upgrade Classification

  • This fields Product Upgrade Classification contains duration of :

    • 3 Months
    • 6 Months
    • 1 Year

  • According to the duration of time selected in Settings, it will upgrade the classification of the Products in product model.
