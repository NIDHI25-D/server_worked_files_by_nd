Stock Inventory Extended
------------------------

##### USAGE

* Custom number functionality in invoice connected with landed cost custome number
* set accounting date in journal entry same as invoice date
* message post functionality in product pricelist
* customer and crm tags in sale report
* Here add relation between Landed Cost and Purchase Order and add a button to the purchase order of landed cost if there are any landed cost are available. 
* Responsible person and track warehouse functionality in transfer, also there are a tab Warehouse there are a various groups are available like OPERATION BEGINNING, AUDIT, OPPORTUNITY AREAS.
* Here extend the pivot view for transfer and Landed Cost
* Change in delivery slip , picking operation and invoice report
* there are a menus available **Order Fill Error** and **Transfer Cancellation Reason** and it's records are reflected to the fields **Order Fill Error** and **Cancellation Reason** in the Transfer.

##### CONFIGURATION
* There are given a groups to the buttons and fields.
* if user has a group **Inventory Supervisor** then only Validate the transfer same this thing are apply at barcode side.
* If user has group **Barcode Owner Access** then only visible and filled the Owner option in barcode transfer.
* If user has group **Stock Move Line Deletion** then only deletion writes of stock move line to the respective user.
