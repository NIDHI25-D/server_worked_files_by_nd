SETU Claims From Website
-----------------------------

### USAGE

* This module is used to create customer claim from website.

### Website
* From Invoices Request claim.
* Added menu to see all claims from web.

### Claim
* Request a claim upon invoice.

### Claim State History
* Managing the claim state history into claims.

### Action Type
* Action typs which are configured with claim category.

### Category of Case
* Declaring the claims category.

### Claim stages
* Dynamic claim stages for claim.

### CRE Team
* Service Agent adding the field to retrieve upon a team to set into claim.

### Configuration
* Added address and enable to website claims.

### Contact
* Claim count to show on the web.

### Transfer
* Upon the dates in transfer if claim is configured, then it will update claim stages upon the dates.

### Vendor Charges status
* to identify, the vendor charges status in to claim.
