Configuration

  • there is a field Hide out of stock products if it is checked then only those products are visible in POS which has stocks.
  • here is the field Allow Order When Out-of-Stock if it is not checked then if the stock are 0 then add to line are not allowed and gives the pop-up message which message are write in the Custom message field.
  • same we can adjust it dynamically like we are maintain this quantity with field Deny order when product stock is equal to, means how much quantity you filled here you can allow only that much of quantity for the pos, by default it is 0, means not allowed order when stock are 0 or below 0, you can manually adjust this quantity.
