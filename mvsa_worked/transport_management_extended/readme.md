Transport Management Extended
-----------------------------

USAGE
=========
* To create Shipping from Sale Orders.
* To create Shipping from picking. 
* To create new shipping if Prepared By field of transport delivery order not set then set it other wise in transport order lines(transport delivery orders) are unlink automatically.

CONFIGURATION
=============
* Set User Configuration:
### Users > Access Rights > Other > Transport Management (Officer,Manager)
### In above config if you set Manager then you are able to see configuration menu.

* to Update Customer Confirmation Date
### Users > Access Rights > Other > Customer Confirmation(confirm sale order)
