Setu Customer Lock
---------------

##### Functionality

    This module use for Lock customer by specific reason to not create sale order and also repair if Locked Customers 
    field are checked in partners and this field are checked from the Is it for customer locking? in Contact Tag, 
    if those tags are set in the partners where Is it for customer locking? are set then it set to the partners and 
    lock those partners and do not allow to create sale orders and repair orders and also POS order.

##### Configuration
    
    There are the field Is it for customer locking? in Tags and those tags are set to the customer where Is it for 
    customer locking? is True then the field Locked Customers in partners are set True and those types of customers 
    are not allowing to the creates Sale orders from both of the sides and also repair order and pos order are 
    not allowing to creates.
