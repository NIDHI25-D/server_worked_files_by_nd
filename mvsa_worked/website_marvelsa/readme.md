Custom Website Sale module for Marvelsa Company
-----------------------------------------------

### USAGE
* automatically delete Abondends carts based set days by user( days set in configuration). 
* To see Refacciones and Accesorios on product page of website.
* To show Customer purchases goal on website on /my/home page.
* To calculate "Customer monthly purchases" and "Fulfillment percent" based "Customer purchases goal".

Configuration
=============
* Set Default SAT Code :
### General Settings > Loyalty Product SAT Code > Default SAT Code

* set Days Configure for Abondends carts deleting cron : 
### Sales > Quotations & Orders > Days Configure for Abondends carts deleting cron

* To see Refacciones and Accesorios on product of shop page of website :
### add values in field spare_parts_product_ids and accessory_product_ids of product
