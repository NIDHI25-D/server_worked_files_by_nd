Setu User Login Activity
-----------------------------

### USAGE

* This module is used to discount line set as per invoiced quantity.
* Configure the invoiced E-wallet line as per configuration into tax.
* If any loyalty program gets archive, then active the product.
* Create the e-wallet lines as discount lines means assigning the e-wallet points as per line with the different types of tax. 
* As per sale order lines, the remaining amount and tax will assign the number of points to that line.
* Return the dict of e-wallet line to be created.
* Check whether the current sale order is the same as domain then apply reward else not.
