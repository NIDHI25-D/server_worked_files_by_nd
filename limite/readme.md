Limit
-----------------------------

### USAGE

* This module is used to limit the customer purchase through setting the credit Limit in Customer.
* Also, if you want to over limit to customer, then Allow Over Credit into customer.

### CONFIGURATION
* To access this feature :
#####           Contacts > Customer > Credit Limit
* Set the Credit Limit into Customer.
#####           Contacts > Customer > Allow Over Credit
* Set Allow Over Credit into Customer.
